# Third-party notices

This repository does not vendor third-party tools or application files.

Tools expected at build time:

- Android Debug Bridge (`adb`) from Android SDK Platform Tools.
- Java runtime.
- apktool.
- uber-apk-signer or an equivalent local APK signer.

The original Android application, extracted APKs, decompiled smali/resources,
rebuilt APKs, split APKs, signing keys, and signer jars are intentionally not
included. Their licenses are separate from this repository.
